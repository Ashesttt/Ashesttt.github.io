# ipad玩Linux没有esc怎么办

# 用  control  +  【

![v2-65edf66d4f7b2a146a899a0782963d35_r.jpg](https://s2.loli.net/2023/04/03/6VTjA8rqiXv5tz3.jpg)

在Linux中，如果您没有ESC键，可以使用`Control + [`来代替。这个组合键实际上等同于按下ESC键。
